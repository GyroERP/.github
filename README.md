# GyroERP `.github`

This repository powers the public [GyroERP organization profile](https://github.com/GyroERP)
and shared GitHub defaults for the GyroERP open organization.

## Contents

| Path | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | Organization landing page shown on [github.com/GyroERP](https://github.com/GyroERP) |
| [`PROJECT.md`](PROJECT.md) | Standard ticket and GitHub Projects workflow |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE/) | Org-wide issue templates (Bug, Feature, Task, Epic) |
| [`.github/workflows/ci.yml`](.github/workflows/ci.yml) | Validates profile and repository scaffolding |
| [`.github/CODEOWNERS`](.github/CODEOWNERS) | Default code owners for reviews |
| [`scripts/`](scripts/) | Branch protection JSON templates |

## Product repositories

- [backend](https://github.com/GyroERP/backend) — Django REST API
- [frontend](https://github.com/GyroERP/frontend) — Web UI
- [database](https://github.com/GyroERP/database) — Schemas and migrations
